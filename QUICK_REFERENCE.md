## 📱 How to Use

### Share Location Once:
```
1. Open "Live Location" page
2. Wait for GPS (shows coordinates)
3. Click sharing button
4. Select contacts
5. Send
```

### Continuous Tracking:
```
1. Open "Live Location" page
2. Click "Start Auto-Share"
3. Confirm
4. Location shared every 2 minutes
5. Click "Stop" when done
```

### Find Police:
```
1. Open "Police Stations" page
2. Auto-search starts
3. View nearby stations
4. Click "Get Directions"
5. Navigate with Google Maps
```

### Emergency SOS:
```
1. Press SOS button (3 seconds)
2. Alarm sounds
3. Choose WhatsApp/SMS
4. Send to all contacts
5. Help on the way!
```
